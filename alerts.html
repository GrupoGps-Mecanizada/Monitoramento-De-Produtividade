📖 Instruções de Configuração

1. O sistema usa o repositório "Monitoramento-De-Produtividade" para centralizar dados

2. Regras são salvas em data/alert-rules.json

3. Alertas gerados ficam em data/alerts-generated.json

4. Sistema usa JSONP para evitar problemas de CORS

5. Sistema detecta automaticamente novos alertas sem duplicação


🔐 Como Configurar o Token GitHub:
  1. Acesse: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Clique em "Generate new token (classic)"
  3. Nome: "Sistema Alertas GrupoGPS"
  4. Selecione: repo (Full control of private repositories)
  5. Clique em "Generate token"
  6. Copie o token (ghp_xxxx...) e cole no campo acima
  7. Clique em "Salvar Configurações"

✅ Segurança: Token nunca fica no código público + IDs únicos + Performance melhorada
