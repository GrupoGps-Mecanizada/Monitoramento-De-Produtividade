📖 Instruções de Configuração

1. Crie um repositório no GitHub para armazenar as regras (ex: Sistema-Alertas)

2. Gere um Personal Access Token no GitHub com permissões de "repo"

3. Preencha as configurações acima com os dados do seu repositório

4. Teste a conexão para verificar se está funcionando

5. Após configurar, todas as regras serão sincronizadas entre dispositivos


⚠️ Nota: O token é armazenado localmente no navegador. Para máxima segurança, use um token com permissões mínimas apenas para o repositório específico.
